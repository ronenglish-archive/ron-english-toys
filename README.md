# Ron English Toys

Starter GitHub Pages repository for Ron English designer toys, custom toys, figure editions, and related Trampt records.

Public GitHub Pages URL once enabled:

https://ronenglish-archive.github.io/ron-english-toys/

Initial structure:

- index.html - repository splash page
- toys/index.html - toys landing page
- toys/designer-toys/index.html
- toys/custom-toys/index.html
- toys/sculptural-editions/index.html
- toys/a-z/index.html
- prints/index.html
- original-art/index.html
- review/index.html
- css/styles.css
- images/Trampt/ - recommended first location for the extracted named images
- images/Web-Documentation/
- data/source-spreadsheets/
- data/generated/

Important workflow note:

Do not dump all 1,000+ records onto the public landing page. Use category landing pages, an A-Z index, and generated individual item pages.

The .gitkeep files only exist so Git tracks empty folders before images/data are added.
